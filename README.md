# Chessbot

Bu proje, bir satranç botu uygulamasıdır.

## Kurulum

Projenin bağımlılıklarını yüklemek için aşağıdaki komutları kullanın:

```bash
npm install
```

## Çalıştırma

Uygulamayı başlatmak için:

```bash
npm start
```
